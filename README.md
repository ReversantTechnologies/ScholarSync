# ScholarSync  
### Your Smart Academic Companion

ScholarSync is a **fully offline academic companion app** developed by **Reversant Technologies**, built specifically for college students to **track, analyze, and improve academic performance**â€”with **no ads, no internet dependency, and full privacy**.

## Features

### CGPA Calculator (Anna University)
- Accurate CGPA calculation based on Anna University regulations  
- Semester-wise and cumulative GPA tracking  

### Internal Marks Management
- Store and manage internal assessment marks  
- Quick access during exams and reviews  

### Course Status Tracking
- Track completed, ongoing, and pending courses  
- Clear visibility of academic progress  

### Document Organizer
- Secure offline storage for academic documents  
- Easy access anytime, anywhere  

### Performance Visualization
- Visual representation of academic progress  
- Identify strengths and areas for improvement  

### Future Performance Prediction
- Predict future academic performance using past data  
- Helps in academic planning and goal setting

## Architecture Overview

- **Flutter** for cross-platform UI  
- **GetX** for:
  - State management  
  - Dependency injection  
  - Route management  
- **Hive** for:
  - Fast, lightweight local data storage  
  - Structured academic data handling  

All data is stored **on-device only**, ensuring **maximum privacy and performance**.
